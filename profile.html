April 15

Unity, the Open Doc Lab and VR/AR@MIT are hosting a FREE Unity workshop for all skill levels. Unity rep Francis Duranceau will be presenting. The Unity workshop is offered in conjunction with theHackingVR class as part of the Oculus NextGen program.

Attendees will be introduced to building VR games and experiences in Unity while learning, hands on, about the workflows and features used by top game studios. During this training, attendees will learn how to build a brand new 3...D game and 3D experience start to finish while touching upon many of the diverse systems and tools that Unity offers. Material to be covered include:

- General workflows and VR build settings
- Unity scripting to create inactions
- Global illumination (GI) to create realistic light
- Physically based rendering (PBR) to create stunning graphics
- Adding audio effects in VR
- Animation (Mecanim) for animation of your character
- VR 360 image with 3D interaction

Important Instructions:
Make sure you have AT LEAST Unity Personal downloaded and installed - https://store.unity.com/download?ref=personal

Instructions on setting up Android development with Unity can be found - https://docs.unity3d.com/Manual/android-sdksetup.html

Watch the following video as a primer -
https://www.youtube.com/watch?v=5nvjty1g87I&feature=youtu.be

This workshop is designed for participants that are familiar with Unity. If youre arent familiar with Unity, you can still attend, but please be sure to complete this tutorial - https://www.youtube.com/watch…
